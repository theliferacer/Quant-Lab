# Quant-Lab 🧪
Quant-Lab is a dedicated space for implementing and benchmarking core quantitative finance algorithms. This repository serves as a technical portfolio, showcasing the application of mathematical theory to financial engineering problems.

🚀 Featured Projects

Variance Reduction (Control Variates): Optimized Monte Carlo simulations to reduce standard error and improve convergence speed for option pricing.

Delta-Neutral Hedging (In Progress): Simulating dynamic hedging strategies to manage Greek risks in a Black-Scholes environment

Monte Carlo Risk Engine: A C++ based simulation framework for asset price paths using Geometric Brownian Motion (GBM).

🛠️ Tech Stack

Languages: C++ (Performance & Simulation), Python (Data Analysis & Visualization)

Focus Areas: Stochastic Calculus, Risk Neutral Valuation, Numerical Methods, and Greeks Management.
